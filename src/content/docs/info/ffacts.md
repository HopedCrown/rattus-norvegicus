Fun fact 1:  
Norway rats are so common a pest, that you can get rodenticides (pesticides for rodents) basically **anywhere**!  
Fun fact 2:  
It is very simple to trap and bait these rats, as they will gnaw through the surrounding paper, so keeping some rice in the same bag as some rodenticide can (and will) kill them.  
Fun fact 3:  
They have a social hierarchy determined by size and gender (a patriarchic society with size as a big factor).
